# DesafioPay 🚀

**Pare de apenas observar a Web3. Comece a conquistá-la.**

O **DesafioPay** é uma arena de desafios Web3 *mobile-first* que redefine a gestão de tarefas transformando onboarding em entretenimento, comunidade em progresso e curiosidade em valor usando ativos digitais reais.

Este projeto nasceu no **Tech Innovation Hub • Senac Chapada • Manaus-AM**.

---

## 🎯 Pilares do Projeto

O ecossistema é alicerçado em 4 pilares fundamentais de impacto:

1. ⚡ **Resolver o Engajamento:** Transformamos a passividade em missões ativas aplicando gatilhos de recompensa para retenção de usuários.
2. 🛡️ **Transparência Imutável:** Toda missão e recompensa é registrada on-chain (blockchain), garantindo que os dados e tokens gerados sejam 100% autênticos e seguros.
3. 🌳 **Escalabilidade de Elite:** Arquitetura criada para um crescimento exponencial e para processar múltiplas transações simultâneas.
4. 🎁 **Incentivos Digitais:** Funciona sob a economia do Token Nativo **DSP**, crescendo em valor conforme a expansão da rede.

## 🛠️ Tech Stack

A infraestrutura de elite do DesafioPay é impulsionada por tecnologias modernas:

- **Frontend:** HTML5, Tailwind CSS
- **Backend/API:** Python, Flask
- **Banco de Dados:** SQLAlchemy
- **Web3:** Integração e validação de tokens on-chain e geração de wallets

## ⚙️ Como Funciona a Arena?

1. **Forje sua Identidade:** Gere sua carteira Web3 no ecossistema instantaneamente.
2. **Aceite o Desafio:** Cumpra missões diárias com validação imediata e acompanhamento de níveis.
3. **Conquiste a Blockchain:** Tarefas reais registradas na rede transformam as metas em progresso público.
4. **Reivindique o Tesouro:** Ganhe reputação interna (off-chain) e tokens valorizados DSP (on-chain).
